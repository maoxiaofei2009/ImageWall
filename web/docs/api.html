Uploadaj novu sliku na server. Kao response vraća informacije o uploadanoj slici.

Parametri:

  • image - slika koja se želi poslati
  • description - opis slike (opcionalno)
  • tag - tag koji je vezan na sliku (opcionalno)
  • lat - latitude lokacije (opcionalno)
  • lon - longitude lokacije (opcionalno)
